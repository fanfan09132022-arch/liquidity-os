# LiquidityOS V2 产品需求文档 v1.1

> 状态：需求锁定 ✅ | 更新日期：2026-03-01

## 一、产品目标

两个核心目标（所有功能围绕这两点）：
1. **推特涨粉** @partrick2022 —— 每日输出有价值的研判内容
2. **个人学习成长** —— 建立可重复执行的宏观交易框架

## 二、四层流动性框架（核心理论）

| 层级 | 名称 | 定义 |
|------|------|------|
| L1 | 宏观流动性 | Global Net Liquidity = 美联储资产负债表 - TGA账户余额 - 逆回购(RRP)余额 |
| L2 | 稳定币购买力 | USDT 市值趋势，市场实际可用的子弹数量 |
| L3 | 链上偏好 | Solana vs Ethereum vs BSC TVL，判断热钱聚集在哪条链 |
| L4 | Meme 个股流动性 | 具体代币的资金结构、持仓结构、换手率 |

逻辑：**从宏观到微观逐层过滤** —— L1 决定有没有水 → L2 决定子弹够不够 → L3 决定战场在哪 → L4 决定具体哪个币

## 三、Section 1：流动性指标

### 3.1 四层指标模块

每层独立「🔄 更新」按钮，AI 通过 web_search 自动抓取。

**L1 宏观流动性**
- Global Net Liquidity 数值
- 趋势方向（扩张↑ / 收缩↓ / 横盘→）
- AI 一句话摘要

**L2 稳定币购买力**
- 稳定币总市值（defillama）
- USDT 市值 + 趋势
- 稳定币占比 %
- 自动提示：扩张→🟢"子弹充足" / 收缩→🔴"控制仓位"

**L3 链上偏好**
- Solana / Ethereum / BSC 稳定币流量对比
- AI 推荐主战场

**L4 市场情绪**
- 恐惧贪婪指数 0-100（alternative.me）
- 自动映射状态

### 3.2 顶部信号灯（始终可见）

四个信号灯：🌐 宏观 / 💰 稳定币 / ⛓️ 链上 / 😨 情绪

自动综合结论：
- 4🟢 → "全面积极，可重仓出击"
- 3🟢1🟡 → "整体乐观，正常仓位"
- 含1🔴 → "谨慎防守，等待信号"
- 3🔴+ → "全面撤退，现金为王"

今日关键判断（手动输入，流向推文草稿）

## 四、Section 2：Meme 战壕

### 4.1 Top 50 Meme 市值榜
字段：排名 / 名称 / 链 / 发行时间 / 市值 / 24h涨跌
「+」按钮加入研判区，AI 一键抓取

### 4.2 热门币种（7日）
三列并排（Sol / BSC / Base），每链 5-8 个
字段：名称 / 链 / 市值 / 7d% / 热点来源

### 4.3 新币（24h）
三列并排，每链 3-5 个
字段：名称 / 链 / 上线时间 / 初始市值 / LP比

### 4.4 研判区（核心）

最多 5 个币，每个研判卡包含：

**① 基本信息**：$TICKER [链标签] / 市值 / 24h% / 添加时间

**② 叙事**：AI 自动填写一句话，用户可编辑

**③ 资金结构**（自动抓取）
- LP 比：>10%🟢 / 5-10%🟡 / <5%🔴
- 换手率：0.5-2x🟢买入 / <0.5🟡观察 / 2-5🟡追涨 / >5🔴过热
- 流动池大小

**④ 持仓结构**
- Solana：Helius API 自动抓取
- BSC/Base：AI 搜索
- Top10持仓%：<30%🟢 / 30-50%🟡 / >50%🔴
- 开发者状态：持有中 / 已减仓 / 已清仓
- 聪明钱数量 + GMGN 链接

**⑤ 社区**：Twitter 搜索 / TG 链接

**⑥ KOL 提及**（每币独立「🔍 搜索 KOL」按钮，动态搜索）

**⑦ 我的评分**（加权）
- 叙事 40% / 资金 25% / 持仓 20% / 社媒 15%
- 每维度：🟢强(3) / 🔵可(2) / 🟠弱(1) / 🔴差(0)

**⑧ 综合结论**（自动）
- ≥2.5 → ✅ 值得关注，考虑建仓
- 1.8-2.5 → 👀 继续观察
- 1.0-1.8 → ⚠️ 综合偏弱，暂时跳过
- <1.0 → ❌ 不建议介入

## 六、数据持久化

- `liquidity:YYYY-MM-DD` — L1-L4 数据
