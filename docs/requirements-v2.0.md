## 一、项目背景

### 1.1 我是谁，在做什么

Twitter 账号 @partrick2022，专注 Solana Meme 币交易。  
核心目标：**推特涨粉 + 个人交易能力成长。**  
所有工具的开发，都服务于这两个目标。

### 1.2 为什么需要这个看板

目前的工作流程：  
每天手动打开 5-8 个网站（MacroMicro、DeFiLlama、DexScreener、GMGN、Twitter）→ 分别查看数据 → 在脑子里拼图 → 做出判断。

**问题：**  
- 信息分散，每次耗时 40 分钟以上
- 没有统一的框架强制自己按顺序思考
- 好的判断过程没有被记录，无法复盘
- 无法生成结构化的推文草稿

**解决方案：**  
开发一个整合所有数据源、按四层流动性框架组织的交易看板，把每日研判流程标准化，同时自动生成推文草稿。

## 二、核心概念：四层流动性框架（含 L0）

> 这是看板的理论基础，所有功能模块都围绕它设计

### L0：宏观周期定位（框架的框架）

**含义：** 在读任何具体数据之前，先确认自己在整个流动性大周期的哪个位置。这决定整体仓位偏见。

**组成：**
- **L0-A：全球流动性大周期阶段**（Howell 65 个月周期：扩张期 / 见顶区 / 收缩期）
- **L0-B：BTC 周期位置**（BTC 现价 / 200 日均线比率 + MVRV Z-Score）

**读法：** L0 是"季节"，其他层是"天气"。先知道现在是冬天还是夏天，再决定穿什么衣服。

---

### L1：全球净流动性

**公式：** 美联储资产负债表 − TGA 账户余额 − 逆回购（RRP）余额  
**含义：** 真正流向金融市场的净流动性，是所有风险资产最终极的驱动变量。  
**关注点：** 增速变化（动量），不是绝对值。连续扩张 3 周 = 早期信号。

**⚠️ 时滞警告：** L1 变化领先 BTC 价格约 13 周（5-16 周区间），领先 Meme 币更长。  
看 L1 时必须标注"已持续第几周"，而不是只看当下快照。

---

### L2：稳定币弹药（修正版）

**错误用法（旧）：** 只看 USDT 市值  
**正确定义：**

- **L2-A（全局弹药）：** Total Stablecoin Market Cap 7 日净变化  
  = Δ（USDT + USDC + USDe + FDUSD + DAI + 其他）过去 7 天  
  数据源：DefiLlama → Stablecoins  

- **L2-B（链上弹药）：** Solana 链上稳定币净流入（7 日）  
  数据源：DefiLlama → Chain 对比  

**为什么这样改：** 2026 年 USDT 受 MiCA 监管压力市值缩减，USDC 年增 72%，只看 USDT 会误读成"流动性收缩"，实际是稳定币结构迁移。需要看总量。

---

### L3：链上资金偏好

**含义：** 流动性到了链上，去了哪条链？决定战场。  
**组成：**  
- Solana vs Ethereum vs BSC TVL（存量，DefiLlama）  
- Solana 链上稳定币净流入 vs 其他链（增量，更灵敏）

---

### L4：Meme 个股流动性

**含义：** 具体代币的资金结构，决定单个标的是否值得介入。  
**组成：**  
- 持仓结构（头部地址集中度）  
- LP 比例（流动性池规模 / 市值）  
- 换手率（24h 交易量 / 市值）  
- 聪明钱动向（GMGN）

---

### 时滞叠加读法（重要）

```
正确读图顺序：

Step 1：L0 → 大周期在哪？（顺风 / 逆风？）
Step 2：L1 → 水龙头方向，已持续几周？
Step 3：L2-A + L2-B → 弹药到位了吗？
Step 4：L3 → 战场在 Solana 吗？
Step 5：L4 → 最后才看具体标的
```

## 四、核心功能模块

### 模块 A：宏观信号仪表盘（L0-L3）

**功能描述：**  
展示从 L0 到 L3 的所有宏观指标，每个指标有信号灯（🟢/🟡/🔴）和简短说明。  
顶部有一个"系统综合信号"卡片，综合四层得出当日总判断。

**具体指标清单：**

| 层级 | 指标名称 | 数据来源 | 更新频率 |
|------|---------|---------|---------|
| L0-A | 全球流动性大周期阶段 | 手动输入（每月更新） | 月度 |
| L0-B | BTC/200日均线比率 | AI 搜索 / 手动输入 | 日 |
| L0-B | MVRV Z-Score | CryptoQuant / 手动输入 | 日 |
| L1 | 美联储资产负债表 | MacroMicro / 手动输入 | 周 |
| L1 | TGA 账户余额 | MacroMicro / 手动输入 | 周 |
| L1 | 逆回购余额（RRP） | MacroMicro / 手动输入 | 周 |
| L1 | GNL（计算值） | 自动计算 | 周 |
| L1 | GNL 连续扩张周数 | 手动输入 | 周 |
| L2-A | 稳定币总市值 7日净变化 | AI 搜索 / 手动输入 | 日 |
| L2-B | Solana 链上稳定币净流入（7日） | AI 搜索 / 手动输入 | 日 |
| L3 | Solana TVL | DefiLlama / AI | 日 |
| L3 | Ethereum TVL | DefiLlama / AI | 日 |
| L3 | BSC TVL | DefiLlama / AI | 日 |

**数据输入方式（V2 阶段）：**
- 每个指标旁有「🔄 AI 查询」按钮，点击后 Claude AI 自动搜索并填入
- 用户可以手动修正任何字段
- 支持手动直接输入（AI 挂了也能用）

**信号灯判断规则（每层）：**

| 层 | 🟢 绿（看涨） | 🟡 黄（中性） | 🔴 红（看跌） |
|----|------------|------------|------------|
| L0-A | 扩张期 | 过渡期 | 收缩期 |
| L0-B | BTC > 200日均 × 1.0 | 接近均线 ±5% | BTC < 200日均 |
| L1 | GNL 连续扩张 ≥3 周 | 扩张 1-2 周 | 收缩或停滞 |
| L2 | L2-A 和 L2-B 均净增 | 仅一项净增 | 均净减 |
| L3 | Solana TVL 7日增速最高 | 持平 | Solana 资金外流 |

**系统综合信号计算：**  
5 层信号灯 → 🟢=1分 / 🟡=0.5分 / 🔴=0分 → 总分 ÷5  
- ≥0.8：🔵 进攻  
- 0.6-0.8：🟢 积极  
- 0.4-0.6：🟡 观望  
- <0.4：🔴 防御

### 模块 B：24H 热点新币研判（L4）

**功能描述：**  
从 DexScreener 热榜获取 Solana 新币，对每个候选币进行四维研判，生成综合评分和推文草稿。

**子功能 B1：AI 自动抓取热榜**  
- 点击「🔄 抓取热榜」按钮
- Claude AI + Web Search 自动搜索 DexScreener Solana 热榜
- 返回 8 个候选币，展示：市值、流动性池、LP 比、换手率、24h 涨跌
- 每个币有「＋」按钮，加入研判列表（最多 5 个）

**子功能 B2：研判卡**  
每个加入的币，生成一张研判卡，包含：

```
── 基础数据行 ──────────────────────────────
市值 / 流动性池 / LP比 / 换手率 / 24h交易量 / 24h涨跌

── 一键跳转链接 ──────────────────────────────
🐋 GMGN 聪明钱 | 🐦 Twitter 热度 | 📊 DexScreener

── 四维研判（每维：评分按钮 + 文本备注）──────────
💡 叙事/题材  → 🟢强 / 🔵可 / 🟠弱 / 🔴差 + 备注
🔗 链上数据  → 🟢强 / 🔵可 / 🟠弱 / 🔴差 + 备注
🐋 聪明钱动向 → 🟢强 / 🔵可 / 🟠弱 / 🔴差 + 备注
📣 社媒热度  → 🟢强 / 🔵可 / 🟠弱 / 🔴差 + 备注

── 加权综合评分 ───────────────────────────────
叙事 40% + 链上 25% + 聪明钱 20% + 社媒 15%
→ 综合结论：✅值得关注 / 👀继续观察 / ⚠️暂时跳过 / ❌不建议
```

**子功能 B3：推文草稿生成**  
有 ≥2 个研判卡时，自动生成今日推文草稿：

```
格式示例：
今日热点研判 🧭 3/2

L0-B BTC 67K·低于200MA | L2 稳定币净减 $1.2B | 逆风环境

$TOKEN1  ✅ 值得关注
   🟢叙事  🟢链上  🔵聪明钱  🔵社媒
   💡 [叙事备注前55字]

$TOKEN2  👀 继续观察
   🔵叙事  🟠链上  🟢聪明钱  🟠社媒

——
四层流动性框架 | @partrick2022
#Solana #Meme #加密货币
```

### 模块 C：贪婪恐惧指数

**功能描述：**  
可编辑的数字 + 滑动条可视化，值域 0-100。  
每次打开手动更新当日指数（来源：alternative.me）。

### 模块 D：今日交易笔记

**功能描述：**  
自由文本区域，记录当日市场观察、入场理由、止损位等。  
数据本地持久化，按日期存储（window.storage API）。

---

### 模块 E：历史回顾（只读）

**功能描述：**  
查看过去某天的研判记录（宏观指标 + 研判卡 + 笔记）。  
只读模式，不可修改历史数据。  
入口：顶部导航「📅 历史」按钮。

## 七、分期开发计划

### Phase 1：宏观信号层（独立可用）✦ 最高优先

**目标：** 独立可用的宏观看板，不依赖热榜功能  
**包含：** 模块 A（L0-L3 指标 + 信号灯 + 系统综合信号）+ 模块 C（恐惧贪婪）  
**交付标准：**
- 每层指标可手动输入
- 信号灯自动计算
- 系统综合信号正确显示
- 数据本地持久化，刷新不丢失

---

### Phase 2：热榜抓取

**目标：** AI 自动搜索 Solana 热榜，展示候选币列表  
**包含：** 模块 B 的子功能 B1  
**交付标准：**
- 点击按钮 15-25 秒内返回 ≥6 个候选币
- 展示市值 / LP比 / 换手率等指标
- 可一键加入研判列表

---

### Phase 3：研判卡 + 推文草稿

**目标：** 完整的个币研判流程  
**包含：** 模块 B 的子功能 B2 + B3 + 模块 D（笔记）  
**交付标准：**
- 研判卡四维评分 + 备注完整可用
- 综合评分自动计算
- 推文草稿自动生成，一键复制

---

### Phase 4：历史回顾 + 体验完善

**目标：** 完整产品打磨  
**包含：** 模块 E（历史回顾）+ 移动端体验优化 + 边缘案例处理  
**交付标准：**
- 可查看任意历史日期数据
- 移动端无明显排版问题
- AI 查询失败有清晰提示和手动兜底

## 八、验收标准

每个 Phase 完成后，我按以下标准验收：

1. **功能完整性：** 本 Phase 所有功能点均可正常使用
2. **数据准确性：** 计算值（GNL、综合信号分）结果正确
3. **持久化：** 刷新页面、关闭重开，数据不丢失
4. **移动端：** 在手机浏览器（Safari/Chrome）上显示正常
5. **容错性：** AI 查询失败时，看板仍可手动使用，不崩溃

## 九、已知局限性（诚实记录）

1. **L4 预测能力有限：** Meme 币 50% 以上涨跌是情绪和叙事驱动，宏观只是底色，本工具无法预测单币走势。
2. **传导时滞：** L1→L4 约 13 周传导，本框架是辅助决策工具，不是交易信号系统。
3. **数据延迟：** 部分指标（L1）为周更数据，存在天然滞后。
4. **AI 搜索不稳定：** 网络波动或 DexScreener 反爬时，AI 搜索可能失败，需手动补充。
5. **稳定币覆盖范围：** L2 追踪主流稳定币，小众稳定币（如 PYUSD、RLUSD）未包含，总量有一定误差。
