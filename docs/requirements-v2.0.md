## 一、项目背景

### 1.1 我是谁，在做什么

Twitter 账号 @partrick2022，专注 Solana Meme 币交易。  
核心目标：**推特涨粉 + 个人交易能力成长。**  
所有工具的开发，都服务于这两个目标。

### 1.2 为什么需要这个看板

目前的工作流程：  
每天手动打开 5-8 个网站（MacroMicro、DeFiLlama、DexScreener、GMGN、Twitter）→ 分别查看数据 → 在脑子里拼图 → 做出判断。

**问题：**  
- 信息分散，每次耗时 40 分钟以上
- 没有统一的框架强制自己按顺序思考
- 好的判断过程没有被记录，无法复盘
- 无法生成结构化的推文草稿

**解决方案：**  
开发一个整合所有数据源、按四层流动性框架组织的交易看板，把每日研判流程标准化，同时自动生成推文草稿。

## 二、核心概念：四层流动性框架（含 L0）

> 这是看板的理论基础，所有功能模块都围绕它设计

### 框架整体逻辑
```
L0 → 确定仓位上限和操作偏见（进攻/防守）
  ↓
L1 → 确认流动性方向，判断信号链走到哪一步
  ↓
L2 → 确认弹药到位，战场聚焦在哪条链
  ↓
L3 → 确认 Meme 板块整体热度
  ↓
L4 → 研判具体标的，做出进出场决策
```

**核心原则：** 上层框架是下层操作的前提条件。L0 偏空时，即使 L4 出现强信号，也应缩小仓位规模、提高胜率阈值、快进快出。不要用 L4 的"机会感"去否定 L0 的大背景。

---

### L0：宏观周期定位（框架的框架）

**含义：** 在读任何具体数据之前，先确认自己在整个流动性大周期的哪个位置。这决定整体仓位偏见。

**组成：**
- **L0-A：全球流动性大周期阶段**（Howell 65 个月周期：扩张期 / 见顶区 / 收缩期）
- **L0-B：BTC 周期位置**（BTC 现价 / 200 日均线比率 + MVRV Z-Score）

**读法：** L0 是"季节"，其他层是"天气"。先知道现在是冬天还是夏天，再决定穿什么衣服。

---

### L1：全球净流动性

**公式：** 美联储资产负债表 − TGA 账户余额 − 逆回购（RRP）余额  
**含义：** 真正流向金融市场的净流动性，是所有风险资产最终极的驱动变量。  
**关注点：** 增速变化（动量），不是绝对值。连续扩张 3 周 = 早期信号。

**⚠️ 时滞警告：** L1 变化领先 BTC 价格约 13 周（5-16 周区间），领先 Meme 币更长。  
看 L1 时必须标注"已持续第几周"，而不是只看当下快照。

**信号链触发顺序（顺序相对稳定，时间不固定）：**
```
① L1 开始扩张（Fed 缩表放缓，TGA 开始释放）
↓ 等待 ↓
② 全球稳定币总量开始净增（弹药进场）
↓ 等待 ↓
③ BTC 站稳 200 日均线并开始新高（主赛道确认）
↓ 等待 ↓
④ Solana 链上稳定币净流入开始加速（战场聚焦）
↓ 此时 ↓
⑤ Meme 币交易量开始放大，新币层出不穷（L4 激活）
```

---

### L2：稳定币弹药

**含义：** 监测稳定币数据，判断场内弹药总量是否充足，以及热钱聚集在哪条链。

**组成：**

- **Solana vs ETH vs BSC TVL（总量）**  
  所有主要稳定币的总规模，作为总流动性参考  
  数据源：DefiLlama

- **Total Stablecoin Market Cap 7 日净变化**  
  = Δ（USDT + USDC + USDe + FDUSD + DAI + 其他）过去 7 天  
  作为总流动性参考  
  数据源：DefiLlama → Stablecoins

- **Solana / Base / BNBChain 链上稳定币净流入（7 日）**  
  衡量战场在哪条链  
  数据源：DefiLlama → Chain 对比

**为什么这样定义：** 2026 年 USDT 受 MiCA 监管压力市值缩减，USDC 年增 72%，只看 USDT 会误读成"流动性收缩"，实际是稳定币结构迁移。需要看总量，同时看链上净流入判断战场。

---

### L3：Meme 板块流动性

**含义：** Meme 板块整体是否活跃，决定当前市场是否适合做 Meme 交易。

**组成：**
-Solana DEX 日交易量（单位 B，来源 DeFiLlama）
-Solana DEX 7日前交易量（用于自动计算 7日变化百分比）
-Meme 市值趋势（全链，来源 CoinGecko）
-Base DEX 交易量趋势（辅助，来源 DeFiLlama）
-BSC DEX 交易量趋势（辅助，来源 DeFiLlama）

---

### L4：Meme 个股流动性

**含义：** 具体代币的资金结构，决定单个标的是否值得介入。  
**组成：**  
- 持仓结构（头部地址集中度）  
- LP 比例（流动性池规模 / 市值）  
- 换手率（24h 交易量 / 市值）  
- 聪明钱动向（GMGN）

---

### 时滞叠加读法（重要）
```
正确读图顺序：

Step 1：L0 → 大周期在哪？（顺风 / 逆风？）
Step 2：L1 → 水龙头方向，已持续几周？
Step 3：L2 → 弹药到位了吗？战场在哪条链？
Step 4：L3 → Meme 板块整体热吗？
Step 5：L4 → 最后才看具体标的
```

## 四、核心功能模块

### 模块 A：宏观信号仪表盘（L0-L3）

**功能描述：**  
展示从 L0 到 L3 的所有宏观指标，每个指标有信号灯（🟢/🟡/🔴）和简短说明。  
顶部有一个"系统综合信号"卡片，综合四层得出当日总判断。

**具体指标清单：**

| 层级 | 指标名称 | 数据来源 | 更新频率 |
|------|---------|---------|---------|
| L0-A | 全球流动性大周期阶段 | 手动输入（每月更新） | 月度 |
| L0-B | BTC/200日均线比率 | AI 搜索 / 手动输入 | 日 |
| L0-B | MVRV Z-Score | CryptoQuant / 手动输入 | 日 |
| L1 | 美联储资产负债表 | MacroMicro / 手动输入 | 周 |
| L1 | TGA 账户余额 | MacroMicro / 手动输入 | 周 |
| L1 | 逆回购余额（RRP） | MacroMicro / 手动输入 | 周 |
| L1 | GNL（计算值） | 自动计算 | 周 |
| L1 | GNL 连续扩张周数 | 手动输入 | 周 |
| L2 | Solana / ETH / BSC TVL | DefiLlama / AI | 日 |
| L2 | 稳定币总市值 7日净变化 | DefiLlama / AI | 日 |
| L2 | Solana / Base / BNBChain 链上稳定币净流入（7日） | DefiLlama / AI | 日 |
| L3 | Meme 总市值（全链，单位 B）→ 来源 CoinGecko Meme 分类
| L3 | Solana DEX 日交易量（单位 B）→ 来源 DeFiLlama DEX/Solana
| L3 | Solana DEX 7日前交易量（单位 B）→ 同上，查7天前数据
| L3 | Meme 市值趋势（全链）→ 上升/持平/下降
| L3 | Base DEX 交易量趋势（辅助）→ 上升/持平/下降
| L3 | BSC DEX 交易量趋势（辅助）→ 上升/持平/下降

**数据输入方式（V2 阶段）：**
- 每个指标区块内嵌数据源快捷链接，点击跳转到对应网站查看数据
- 用户查看后手动填入数字，所有字段均支持直接输入
- 数据通过 window.storage 按日期持久化，次日打开自动恢复昨天数据
- 宏观数据变化慢（Fed/TGA/RRP 每周一次，TVL 每天变化<1%），多数字段无需每天改
- AI 自动搜索仅用于热榜抓取（每天变化大、手动做痛苦的场景）

数据源快捷链接清单：
- L0-B：TradingView BTCUSD（BTC现价+200MA）、LookIntoBitcoin（MVRV Z-Score）
- L1：FRED WALCL（Fed资产负债表）、FRED WTREGEN（TGA）、FRED RRPONTSYD（RRP）
- L2：DeFiLlama /chains（三链TVL）、/stablecoins（总量7日变化）、/stablecoins/chains（各链净流入）
- L3：CoinGecko /categories/meme-token（Meme总市值）、DeFiLlama /dexs/chains/solana|base|bsc（DEX交易量）
- F&G：Alternative.me /crypto/fear-and-greed-index/

**信号灯判断规则（每层）：**

| 层 | 🟢 绿（看涨） | 🟡 黄（中性） | 🔴 红（看跌） |
|----|------------|------------|------------|
| L0-A | 扩张期 | 过渡期 | 收缩期 |
| L0-B | BTC > 200日均 × 1.0 | 接近均线 ±5% | BTC < 200日均 |
| L1 | GNL 连续扩张 ≥3 周 | 扩张 1-2 周 | 收缩或停滞 |
| L2 | 稳定币总量净增 且 目标链净流入为正 | 仅一项为正 | 均为负 |
| L3 | Meme市值趋势+Solana DEX交易量均涨（加权≥0.7） | 分歧或持平（0.4-0.7） | 均跌（<0.4） |

**系统综合信号计算：**  
5 层信号灯 → 🟢=1分 / 🟡=0.5分 / 🔴=0分 → 总分 ÷5  
- ≥0.8：🔵 进攻  
- 0.6-0.8：🟢 积极  
- 0.4-0.6：🟡 观望  
- <0.4：🔴 防御

### 模块 B：24H 热点新币研判（L4）

**功能描述：**  
从 DexScreener 热榜获取 Solana 新币，对每个候选币进行四维研判，生成综合评分和推文草稿。

**子功能 B1：AI 自动抓取热榜**  
- 点击「🔄 抓取热榜」按钮
- Claude AI + Web Search 自动搜索 DexScreener Solana 热榜
- 返回 8 个候选币，展示：市值、流动性池、LP 比、换手率、24h 涨跌
- 每个币有「＋」按钮，加入研判列表（最多 5 个）

**子功能 B2：研判卡**  
每个加入的币，生成一张研判卡，包含：
```
── 基础数据行 ──────────────────────────────
市值 / 流动性池 / LP比 / 换手率 / 24h交易量 / 24h涨跌

── 一键跳转链接 ──────────────────────────────
🐋 GMGN 聪明钱 | 🐦 Twitter 热度 | 📊 DexScreener

── 四维研判（每维：评分按钮 + 文本备注）──────────
💡 叙事/题材  → 🟢强 / 🔵可 / 🟠弱 / 🔴差 + 备注
🔗 链上数据  → 🟢强 / 🔵可 / 🟠弱 / 🔴差 + 备注
🐋 聪明钱动向 → 🟢强 / 🔵可 / 🟠弱 / 🔴差 + 备注
📣 社媒热度  → 🟢强 / 🔵可 / 🟠弱 / 🔴差 + 备注

── 加权综合评分 ───────────────────────────────
叙事 40% + 链上 25% + 聪明钱 20% + 社媒 15%
→ 综合结论：✅值得关注 / 👀继续观察 / ⚠️暂时跳过 / ❌不建议
```

**子功能 B3：推文草稿生成**  
有 ≥2 个研判卡时，自动生成今日推文草稿：
```
格式示例：
今日热点研判 🧭 3/2

L0-B BTC 67K·低于200MA | L2 稳定币净减 $1.2B | 逆风环境

$TOKEN1  ✅ 值得关注
   🟢叙事  🟢链上  🔵聪明钱  🔵社媒
   💡 [叙事备注前55字]

$TOKEN2  👀 继续观察
   🔵叙事  🟠链上  🟢聪明钱  🟠社媒

——
四层流动性框架 | @partrick2022
#Solana #Meme #加密货币
```

## 七、分期开发计划

### Phase 1：宏观信号层（独立可用）✦ 最高优先

**目标：** 独立可用的宏观看板，不依赖热榜功能  
**包含：** 模块 A（L0-L3 指标 + 信号灯 + 系统综合信号）+ 模块 C（恐惧贪婪）  
**交付标准：**
- 每层指标可手动输入
- 信号灯自动计算
- 系统综合信号正确显示
- 数据本地持久化，刷新不丢失

---

### Phase 2：热榜抓取

**目标：** AI 自动搜索 Solana 热榜，展示候选币列表  
**包含：** 模块 B 的子功能 B1  
**交付标准：**
- 点击按钮 15-25 秒内返回 ≥6 个候选币
- 展示市值 / LP比 / 换手率等指标
- 可一键加入研判列表

---

### Phase 3：研判卡 + 推文草稿

**目标：** 完整的个币研判流程  
**包含：** 模块 B 的子功能 B2 + B3 + 模块 D（笔记）  
**交付标准：**
- 研判卡四维评分 + 备注完整可用
- 综合评分自动计算
- 推文草稿自动生成，一键复制

---

### Phase 4：历史回顾 + 体验完善

**目标：** 完整产品打磨  
**包含：** 模块 E（历史回顾）+ 移动端体验优化 + 边缘案例处理  
**交付标准：**
- 可查看任意历史日期数据
- 移动端无明显排版问题
- AI 查询失败有清晰提示和手动兜底

---

## 八、验收标准

每个 Phase 完成后，我按以下标准验收：

1. **功能完整性：** 本 Phase 所有功能点均可正常使用
2. **数据准确性：** 计算值（GNL、综合信号分）结果正确
3. **持久化：** 刷新页面、关闭重开，数据不丢失
4. **移动端：** 在手机浏览器（Safari/Chrome）上显示正常
5. **容错性：** AI 查询失败时，看板仍可手动使用，不崩溃

---

## 九、已知局限性（诚实记录）

1. **L4 预测能力有限：** Meme 币 50% 以上涨跌是情绪和叙事驱动，宏观只是底色，本工具无法预测单币走势。
2. **传导时滞：** L1→L4 约 13 周传导，本框架是辅助决策工具，不是交易信号系统。
3. **数据延迟：** 部分指标（L1）为周更数据，存在天然滞后。
4. **AI 搜索不稳定：** 网络波动或 DexScreener 反爬时，AI 搜索可能失败，需手动补充。
5. **稳定币覆盖范围：** L2 追踪主流稳定币，小众稳定币（如 PYUSD、RLUSD）未包含，总量有一定误差。
